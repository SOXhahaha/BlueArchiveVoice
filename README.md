# BlueArchiveVoice

> 蔚蓝档案全角色语音（中 / 日 / 韩）  
> Blue Archive Full Character Voices (JP / CN / KR)

## 文件结构

```plaintext
BlueArchiveVoice/
├── cn/                         # 中文语音文件夹
│   ├── [角色名]/               # 按角色名分类
│   │   ├── [皮肤名]/           # 按皮肤名分类
│   │   │   └── Voice.ogg      # 语音文件（原始命名）
│   │   └── [皮肤名]/
│   │       └── Voice.ogg
│   ├── [角色名]/
│   │   └── ...
│   └── ...
├── jp/                         # 日文语音文件夹
│   └── [同 cn 结构]
└── kr/                         # 韩文语音文件夹
    └── [同 cn 结构]
```
